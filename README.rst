ttbplugin plugin for `Tutor <https://docs.tutor.edly.io>`__
###########################################################

ttbplugin plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/Turn-The-Bus/tutor-contrib-ttbplugin

Usage
*****

.. code-block:: bash

    tutor plugins enable ttbplugin
